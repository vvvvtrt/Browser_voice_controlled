# Браузер с голосвым управлением
Проект создан в 2022 году.

 Данный браузер, по задумке, предназначен для людей, у которых проблемы со зрением, но его можно использовать так же и для обычных пользователей. Браузер был написан на языке программирования Python, использовалась графическая библиотека PyQt5, для создания окна. Для считывания и распознавания голоса использовалась библиотека Speech recognition, для воспроизведения голоса - библиотека pyttsx3. Для получения html кода использовалась библиотека requests, для работы с парсингом html кода - библиотека BeautifulSoup4.
